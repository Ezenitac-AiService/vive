# 푸른연구소 배도라지즙 랜딩 페이지 (Pureun Lab Pear Juice Landing Page)

이 프로젝트는 푸른연구소의 프리미엄 배도라지즙 제품을 홍보하기 위한 반응형 웹 랜딩 페이지입니다.  
자연 그대로의 건강함과 프리미엄 이미지를 강조하는 디자인으로 구성되어 있습니다.

## 📁 프로젝트 구조 (Project Structure)

`pear-juice-landing/` 폴더 내에 소스 코드가 포함되어 있습니다.

```
D:\vive\Landing_Page\pear-juice-landing
├── index.html      # 메인 HTML 구조
├── style.css       # 스타일 시트 (CSS)
└── assets/         # 이미지 및 리소스 폴더
```

## ✨ 주요 기능 및 특징 (Features)

*   **히어로 섹션 (Hero Section)**: 제품의 핵심 메시지("자연의 정수, 오롯이 담다")와 구매 유도 버튼(CTA) 배치
*   **신뢰도 강조 (Trust Badges)**: 흐르는 자막(Marquee) 효과를 통해 주요 특징(무첨가, 100% 국내산, HACCP 등) 전달
*   **성분 소개 (Ingredients)**: 국내산 배와 도라지 등 핵심 원료를 강조하는 카드 레이아웃
*   **라이프스타일 (Lifestyle)**: 아이들도 좋아하는 맛과 건강한 습관을 제안하는 감성적인 섹션
*   **반응형 디자인 (Responsive Design)**: 다양한 디바이스 환경에 최적화된 레이아웃

## 🛠 사용 기술 (Tech Stack)

*   **HTML5**: 시맨틱 태그를 활용한 구조화
*   **CSS3**: Flexbox, Grid 레이아웃 및 애니메이션 효과 (Fade-in)
*   **Pretendard Font**: 가독성 높은 프리미엄 폰트 사용

## 🚀 실행 방법 (How to Run)

1.  `pear-juice-landing` 폴더의 `index.html` 파일을 웹 브라우저(Chrome, Edge 등)에서 직접 실행합니다.
2.  또는 VS Code의 'Live Server' 확장 기능을 사용하여 실시간으로 확인할 수 있습니다.
